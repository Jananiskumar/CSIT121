5. Which CRAP Design Principle does the use? Provide at least one example.

CRAP is contrast, repetition, alignment and proximity. Love meow website has all of it. For the contrast, the text is often in a color that contrasts well with the background, making it easy to read. For the repetition, there is a consistent use of fonts, colors, and layout throughout the site, which helps create a cohesive and unified look. For alignment, the elements on the website are well-aligned. For proximity, related items are grouped together, which helps users understand the relationship between different pieces of content.
