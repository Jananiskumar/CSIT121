Lesson Learned

I observed that when I tested on a web browser, I have to save it first and then refresh, or else it won't update. I tested on different web browsers such as Google Chrome, Edge, and Firefox, and they work very similarly. One of the challenges in this project is stitching all my previous projects together, especially the ones with multiple HTML files. I don't know how to group them together as a single lab, so I decided to separate them into parts. Lab 5 and Lab 8 have two different navigation bars because Lab 5 brings you to different sections of the website, and Lab 8 shows interactivity in the navigation bar, which is important to the lab. I like this project because it shows how my websites have improved from the beginning to the end of the semester.
